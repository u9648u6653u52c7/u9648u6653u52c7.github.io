策略模式

发表于 2017-07-11 | 更新于 2017-08-12 | 分类于 编程
字数统计 499

“策略模式”：定义了一系列的算法并提供运行时动态更改算法的能力，保证了客户端代码的简洁性。

实现方式：

  1. 创建策略接口；
  2. 创建多个策略类继承策略接口并通过不同算法实现接口中声明的方法；
  3. 创建一个消费者类，它必须存储一个策略对象的引用并对外提供一个可修改该引用的方法；
  4. 在执行环境的上下文通过动态的更改消费者对象中存储的策略对象引用来达到动态更改算法的目的；
